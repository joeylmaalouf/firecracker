What is it?

Rainmeter for Ubuntu, done in Python.

Rainmeter is a desktop customization tool for Windows. Users can create desktop skins and widgets and customize them to their hearts' desire.

Firecracker is a fanmade rewrite of Rainmeter for Ubuntu. Just like Rainmeter, Firecracker windows are fully customizable through editing of configuration files, giving users the ability to tweak and edit their windows to their liking. Unlike Rainmeter, we implemented a configuration file generator in the form of a graphical user interface, where users can simply click on the widget and select a few options and have their widgets running in no time!

Why?

This is our final project for SoftDes Spring 2015 at Olin. We enjoy using Rainmeter on Windows, and we found that Ubuntu lacks that functionality, so we decided to add it ourselves. We did this with the powerful Python language with the help of some libraries.

How To Use It

In order to run Firecracker manually:

python2 firecracker.py <config file>

In order to have Firecracker run on startup:

  1. clone this repo to some location .../firecracker
  2. $ cd .../firecracker
  3. put your desired configuration in .../firecracker/skins/example.cfg
  4. $ sudo bash install.sh
  5. relogin to your computer

This process will also install the required dependencies (listed below).

Press Ctrl+M with any widget selected in order to bring up the widget creator, or run python2 firecracker_config_generator.py.
